# App Icons

This directory should contain the following icon files for the PWA:

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`
- `badge-72x72.png` (for notification badge)

## How to Generate Icons

You can generate all required icon sizes from a single high-resolution image using:

1. **Online Tools:**
   - https://realfavicongenerator.net/
   - https://favicon.io/favicon-generator/

2. **Manual Creation:**
   - Create a 512x512 PNG image with your logo/icon
   - Use an image editor or online tool to resize to all required sizes
